# Default Daily Routine

The routine is stored in the `routine_sessions` table and managed via the Settings page.
This document preserves the original seed data for reference if a fresh instance needs to be set up manually.

## Sessions (in order)

1. **Morning** — Bathroom break outside · Breakfast in playpen
2. **Mid-morning** — Environmental training in Oslo — carry or sit on bench (10–15 min) · Calm walk home
3. **Lunch** — Lunch in playpen · Calm cuddle
4. **Afternoon** — Mental training: sit / stay · Solo time in playpen
5. **Dinner** — Dinner in playpen · Calm wind-down
6. **Evening** — Evening snack in playpen · Calm wind-down
7. **Night outing** — Last bathroom break — very calm, no play · Cuddle on lap or floor only

## Timing defaults

- Awake window: 40 minutes
- Nap duration: 90 minutes
- Wind-down warning: 25 minutes before end of awake window
- First wake anchor: 09:00
